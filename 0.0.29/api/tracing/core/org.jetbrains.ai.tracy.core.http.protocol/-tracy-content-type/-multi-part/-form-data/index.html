FormData

object FormData : TracyContentType(source)

Represents a specific multipart/form-data content type.

This object is a concrete implementation of the TracyContentType interface and is used to encapsulate the multipart/form-data MIME type. It can be utilized in scenarios where HTTP requests or responses involve submitting form data in a multipart format.

Key characteristics:

  • The type is defined by the constant value TYPE.

  • The subtype is set to "form-data".

  • Implements a string representation of the MIME type via the asString method.
